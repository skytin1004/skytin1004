### **Co-op Translator (Microsoft Azure OSS)**

Isang opisyal na Azure OSS CLI na nag-a-automate ng **maramihang pagsasalin** para sa Markdown at nilalamang imahe (OCR)  
**Epekto:** Pinapagana ang pagsasalin para sa mga repositoryo pang-edukasyon ng Microsoft na may kabuuang 200k‚òÖ+, na tinitiyak na laging napapanahon ang nilalaman sa 10+ wika.

- Lumago mula sa paunang PoC ‚Üí production-grade na Python CLI
- Nagsama ng Azure OpenAI + Azure AI Vision para sa pagsasalin + automation ng OCR
- Ginagamit ng mga repo tulad ng:
  - [Generative AI for Beginners](https://github.com/microsoft/Generative-AI-for-beginners) (84k‚òÖ)
  - [ML for Beginners](https://github.com/microsoft/ML-for-Beginners) (72k‚òÖ)
  - [AI for Beginners](https://github.com/microsoft/AI-for-Beginners) (37k‚òÖ)
  - [AI Agents for Beginners](https://github.com/microsoft/AI-Agents-for-Beginners) (17k‚òÖ)
  - [PhiCookbook](https://github.com/microsoft/PhiCookbook) (3k‚òÖ)
- Itinatampok sa **Open at Microsoft** OSS showcase
- Nakatanggap ng **Microsoft AI MVP 2025**
